Paso a paso

  • inicializar repositorio local -> Git init
  • ver estado de repositorio -> git status
  • agregar archivos -> git add nombreArchivo.extecion
  • agregar cambios a la historia del repo -> git commit -m "comentario"
  • Cambiar nombre de una rama -> git branch -M nombreNuevaRama
  • agregar Repositorio remoto -> git remote add origin url
  • Traer cambios -> git pull origin nombreRama
  • subir cambios -> git push origin nombreRama
